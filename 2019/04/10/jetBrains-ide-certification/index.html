JetBrains IDE（e.g. IntelliJ IDEA )旗舰版认证安装方法（针对在校学生）

发表于 : Apr 10, 2019  20:49:58   分类于 : 后端开发   文章字数 : 516 字   阅读时长 : 1 分钟

前言：

IntelliJ IDEA是java开发的利器，共有两个版本：社区版、旗舰版。JetBrains的计划（Academic License Program）使得edu学生邮箱可以免费申请使用旗舰版本。

1 注意事项

  • 因为Jetbrains网站在外网，所以可能你在访问时出现无法连接的情况，重新尝试连接即可。
  • 另外Jetbrains每年都在美化更新，可能博客中页面的截图位置与你的页面不一致，大家可以对比着使用，如果有问题可以在博客下方留言，我们一起讨论。
  • Jetbrains的学生账号是通用的，不仅适用IntelliJ IDEA，也适用于Pycharm、PhpStrorm、WebStorm等Jetbrains旗下IDE。

2 Edu学生邮箱申请

1.注册Jetbrains的官方账号：https://account.jetbrains.com/login

2.学生邮箱免费申请：https://www.jetbrains.com/student/

3.登录校园邮箱，查收邮件：

  • 登录自己的校园邮箱，等待接受确认邮件（邮件可能被标记为垃圾邮件）

  • 邮件标题是JetBrains Educational Pack Confirmation，点击下方Confirm Request

  • 收到授权邮件，邮件标题为“JetBrains Student License Confirmation”。点击邮件内的”Activate Educational License”链接，将开启学生授权条款，请阅读后点击同意按钮。此时系统将询问你的 JetBrains Account 帐号，就为文章最开始说的校园邮箱注册的账号。

  • 这就得到了授权，这里也会显示授权时长，一般为一年，授权结束后还可以继续申请。

2.进入校园邮箱，打开新收到的JetBrains邮件：
