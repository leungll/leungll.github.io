This time, you are supposed to find A+B where A and B are two polynomials.

Input Specification

Each input file contains one test case. Each case occupies 2 lines, and each line contains the information of a polynomial:
$K N_1 a_{έΑΜNέΑΜ1}έΑΜέΑΜέΑΜέΑΜ N_2 a_{έΑΜNέΑΜ2}έΑΜέΑΜέΑΜέΑΜέΑο N_KέΑΜ a_{έΑΜNέΑΜK}έΑΜέΑΜέΑΜέΑΜ$
where $K$ is the number of nonzero terms in the polynomial, $NέΑΜ_i$έΑΜέΑΜ and $aέΑΜ_{NέΑΜi}έΑΜέΑΜέΑΜέΑΜ(i=1,2,έΜψ,K)$ are the exponents and coefficients, respectively. It is given that
$1 έΚν K έΚν 10Ύ╝Ν0 έΚν NέΑΜ_KέΑΜ έΑΜ< έΜψ < N_2 έΑΜέΑΜ< NέΑΜ_1 έΚν 1000$.

Output Specification

For each test case you should output the sum of $A$ and $B$ in one line, with the same format as the input. Notice that there must be NO extra space at the end of each line. Please be accurate to 1 decimal place.

Sample Input

2 1 2.4 0 3.2
2 2 1.5 1 0.5

Sample Output

3 2 1.5 1 2.9 0 3.2
